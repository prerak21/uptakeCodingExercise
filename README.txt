In this project there are 2 packages one is "src" and other is "lib"

"lib" contains required library files such as chromedriver etc.
"src" contains 3 packages PageObjects, Tests and TestSetup
TestSetup is for the initial beforetest for browserlaunch
PageObjects- As name depicts it contains required locators for respective webpages.
Tests - It contains all the tests to be performed for individual tests.
